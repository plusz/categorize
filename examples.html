Document Categorization API Demo

Annual Tax Report Analysis

In this example, our API analyzed an annual tax report document and:

  • Identified document type: Tax Return Statement
  • Selected category: "Financial Documents"
  • Suggested additional taxonomies:
    • Tax Documentation
    • Annual Reports
    • Financial Statements
    • Regulatory Compliance

Bank Payment Confirmation

The API processed a bank payment confirmation and:

  • Identified document type: Payment Transaction Record
  • Selected category: "Banking Documents"
  • Suggested additional taxonomies:
    • Payment Receipts
    • Transaction Records
    • Financial Transfers
    • Account Statements
